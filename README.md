# content

Content source repository for [transguide.jp](https://transguide.jp/).

The repository default branch is `edit`, so Pages CMS opens there first for
content contributors. Maintainers review and merge pull requests from `edit`
into `main`, then merge `main` into `live` to publish the site. The public site
rebuild is triggered after approved changes land on `live`.

Technical editors may also edit the Markdown files directly.
