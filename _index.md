## About the project
[transguide.jp](/) is an effort to gather resources for the community and present it in an easily-accessible manner for both native Japanese residents, as well as immigrants from around the world and those who may be interested in living in Japan. We will attempt to keep this page up-to-date over time and accept user suggestions and feedback through [our email](mailto:transguidejapan@gmail.com).

### Community
This project is developed by and for the LGBTQ+ community in Japan. If you would like to contribute to the development of this website in any way, either reach out by email, or fork the open-source repository and start a pull request!
